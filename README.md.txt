# 🌦️ Weather Dashboard (Python)

A simple Python CLI app that fetches real-time weather data using the OpenWeather API.

## 🚀 Features
- Get live weather for any city
- Shows temperature, weather condition, wind speed
- Beginner-friendly project using Python + APIs

## 🛠️ Tech Stack
- Python
- Requests library
- OpenWeather API

## 📌 How to Run
1. Clone repo: `git clone https://github.com/CodeWithAbhinaav/weather-dashboard-python`
2. Install dependencies: `pip install requests`
3. Run the app: `python weather_app.py`
4. Enter any city → see weather details 🌍

## 🔑 API Key Setup
1. Create a `.env` file in the project root.
2. Add your OpenWeather API key:
3. Run the project!
